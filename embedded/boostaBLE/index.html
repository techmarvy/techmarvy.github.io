BoostaBLE

BoostaBLE is an Open Source nRF52832 BLE development board with sensors that runs on a built-in AA battery.

Specifications

  • Nordic nRF52832 BLE SoC with 512k Flash, 64k RAM
  • TPS61098 boost converter
  • SI7006-A20 Temperature/humidity sensor
  • LIS2DH12 Accelerometer
  • RGB LED
  • Keystone 2460 AA battery holder
  • Power switch
  • NFC antenna connection
  • Two 1 x 6 female headers for GPIOs
  • PCB antenna

Getting Started with boostaBLE

BoostaBLE is Open Source Hardware, and we're developing interesting projects around it. To get started, read documentation and learn more, visit the BoostaBLE github repository .
