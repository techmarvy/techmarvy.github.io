BLE Force Sensor

This project involved designing a low power haptic feedback sensor for a clinical application. For the controller, we chose an FCC certified Raytac module with the Nordic nRF52832 SoC. The companion mobile application was developed using the Cordova cross-platform framework. The force/pressure was measures using FSR (Force Sensing Resistor) units.
